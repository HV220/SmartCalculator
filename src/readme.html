Требования к установке

LINUX

• sudo apt-get update
• sudo apt-get install qt5-qmake qtbase5-dev qtbase5-dev-tools libqt5svg5-dev libqt5webenginewidgets5 libqt5webchannel5-dev qtwebengine5-dev

MAC OS

• Установить QtCreator

Установка

Запусить MakeFile в корне проекта

• make install

Для запуска тестов, введите следующую команду

• make tests

Для запуска проверки на стиль, введите следующую команду

• make check

Для формирования отчета по тестам, введите следующую команду

• make gcov_report

Для создания архива, введите следующую команду

• make dist

Для удаления программы, введите следующую команду

• make uninstall

Для очистки от лишних файлов, введите следующую команду

• make clean
